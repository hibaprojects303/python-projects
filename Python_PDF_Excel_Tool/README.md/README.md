# PDF & Excel Automation Tool

## Project Description

This project helps you merge multiple Excel files and extract text from PDF files easily, creating ready-to-use files for analysis.

### Features:

- Merge Excel files into one file: `merged_excel.xlsx`
- Extract PDF texts into a CSV file: `extracted_pdf.csv`
- Easy to use for any folder containing PDF and Excel files

### How to Use:

1. Put all Excel and PDF files in a folder named `files_to_merge`
2. Run the script:

```bash
python pdf_excel_tool.py
```
